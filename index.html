      How this tool works:
      
    
        
  1. Drop a bank statement PDF. All pages are read in your browser. Nothing is uploaded.
    2. 
        
  2. The tool detects the bank, locates the Debits/Credits/Balance column positions in the table header, and reads each transaction by column. Debit vs credit is determined by which column the amount sits in — not guesswork.
    3. 
        
  3. Every transaction is auto-categorized using rules tuned for Baybos's "GAPSLITE" expense narration.
    4. 
        
  4. For each row you can change the category in the dropdown. The tool remembers your choice and applies it next time the same description appears.
    5. 
        
  5. For bulk corrections: download the categorized CSV, fill in the Category column in Excel for any Uncategorized or wrong rows, then click Import master CSV.
