How v6 works:
  1. Drop a GTCO bank statement PDF. All pages are read in your browser. Nothing is uploaded.
  2. The PDF is 90°-rotated under the hood, so the parser flips the axis — Y identifies which column (Debits, Credits, Balance, Remarks, etc.), X identifies which transaction row. Debit vs credit is determined by which column the amount sits in — no guesswork.
  3. Every transaction is auto-categorized using rules tuned for Baybos's GAPSLITE narration.
  4. Change any row's category in the dropdown — your choice is remembered and applied next time the same description appears.
  5. For bulk corrections: download the categorized CSV, fill Category column in Excel for any Uncategorized or wrong rows, then click Import master CSV.
