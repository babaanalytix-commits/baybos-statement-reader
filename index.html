      How this works:

      1. Drop a GTCO bank statement PDF above. The tool reads it directly in your browser.

      2. Every transaction is auto-categorized using rules tuned for Baybos's expense narration (the "GAPSLITE" convention).

      3. If a category is wrong, change the dropdown in the Category column. The tool remembers your choice — next month the same description gets your category automatically.

      4. Download a categorized CSV for your accounts, or a summary CSV for monthly reporting.
